# CakeTool

Tool/decryption tool for .cak archives (WIP).

## Support

* v6.7
* v6.8
* v8.1 (use force no enc flag)
* v8.2
* v8.3
* v8.7
* v9.1
* v9.2
* v9.3

This repo is only provided for reference/documentation. No support will be provided.
